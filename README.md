# PortfolioManagerForStocks_BE
A simple implementation of an H2 database. 
The program has the following features:
  - Create a portfolio
        - Add stocks to the portfolio
        - Delete stocks from the portfolio
        - Edit portfolios
        - Delete portfolios
        - Specify currency of portfolio
        - Display total value of the portfolio in the currency specified
  - Add stocks to the "market"
        - Edit stocks
        - Delete stocks
        - Specify the currency, symbol, price of the stock
  - Add Foreign Exchange (FX) Spots to the "market"
        - Edit FX spots
        - Delete FX Spots
        - Specify the currency exchanges and the rate of exchange
